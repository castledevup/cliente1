Litebox, Parallax and top map parameters

It's so easy to edit transition paramaters, open up the file scripts.js and change the following code: 

// Litebox
	$('.litebox').tosrus({
		buttons: 'inline',
		pagination	: {
		add			: true,
		type		: 'thumbnails'
		}
	});
				
				
	// Parallax effect
	$.stellar({
		horizontalScrolling: false,
		verticalOffset: 0,
		responsive:true
	});
			
			
	// Top Map		
	$('.top-bar .map').click(function() {
	$('.map-container').addClass("show");
	});
	
	$('.map-container .close-btn').click(function() {
	$('.map-container').removeClass("show");
	});
